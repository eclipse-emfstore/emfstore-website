EMFStore - a model repository for EMF

EMFStore is a model repository for EMF. The EMFStore-Website is currently moving to Eclipse. Until this is finished, the links will point to the old location: emfstore.org

Das Gruendungsprojekt Model Repository wird unterstuetzt durch das Foerderprogramm EXIST-Gruenderstipendium des Bundesministeriums fuer Wirtschaft und Technologie (BMWi), kofinanziert aus dem Europaeischen Sozialfonds.

Current Status

EMFStore has been hosted on Google Code for three years and is ready to use. EMFStore has recently become an Eclipse project. We are currently preparing the IP Process.
