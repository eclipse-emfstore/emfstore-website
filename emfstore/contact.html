Contact

Mailinglists and Newsgroup

Please post any questions about the EMFStore in the EMFT Eclipse newsgroup . If you have feature requests, questions, bugs or if you want to discuss something, you can send an email to emfstore-dev@eclipse.org; all EMFStore framework developers are subscribed to this mailing list

Bug Reports and Feature Requests

You can report bugs and feature requests in the bugzilla of Eclipse . Also you can report bugs and feature requests to the developer mailinglist .

Twitter

We announce import news such as releases on Twitter: http://twitter.com/emfstore . We always use the #emfstore tag.
