What is EMFStore and why should I use it?

EMFStore is a model repository (server) for EMF featuring collaborative editing and versioning of models. Existing versioning systems such as SVN or GIT are focused on textual artifacts and do not work well for models. The EMFStore is a repository and version control system for the Eclipse Modeling Framework (EMF) designed specifically for models. Instead of versioning the textual serialization of a model, it allows semantic versioning of the model and as a result, supports merging and conflict detection more effectively.
