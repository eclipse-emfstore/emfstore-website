Getting Started

1. Download

Before downloading, be sure to have the latest modeling edition of Eclipse installed since EMFStore will be installed into that instance. You may download EMFStore via the Eclipse Marketplace, as an Update site or as an p2 update site.

Photo from stevendepolo on Flickr
Via Marketplace
Opening Market place
Please copy this URL and follow the instructions:
http://download.eclipse.org/emf-store/milestones/100RC5
  1. Navigate to Help → Install New Software
  2. Paste the copied URL into the text field label with "Work with" and press enter
  3. Select "EMFStore Core Features" and press next
  4. Accept the license agreement and finish the installation

Download started

2. Download the example model

You can use the "Make It Happen" example model, which can be retrieved from this repository:

git://github.com/eclipsesource/makeithappen.git

You can also create the "Make It Happen" example model from scratch, or you can create a custom model for your own application; a general description on how to do that can be found here.

3. Get to know EMFStore better

Read about the features of EMFStore and learn how to configure EMFStore. For detailed information, read the documentation of EMFStore.
