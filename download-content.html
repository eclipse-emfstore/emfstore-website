All downloads are provided under the terms and conditions of the Eclipse Foundation Software User Agreement unless otherwise specified. To install EMFStore, you need a running version of Eclipse. The simplest way to get started is to use the latest modeling edition.

1.6.x Service Releases

Please copy this URL:
http://download.eclipse.org/emf-store/releases_16
  1. Navigate to Help → Install New Software
  2. Paste the copied URL into the text field label with "Work with" and press enter
  3. Select "EMFStore SDK" and press next
  4. Accept the license agreement and finish the installation

To get started follow these simple three steps.
